office2016的安装

首先，下载迅雷（免安装版，取自落尘之木），解压，然后在Program里打开Thunder.exe，然后添加任务，复制如下链接粘贴至下载框中

ed2k://|file|SW_DVD5_Office_Professional_Plus_2016_64Bit_ChnSimp_MLF_X20-42426.ISO|1123452928|31087A00FF67D4F5B4CBF4AA07C3433B|/

下载完成后，打开set-up.exe.选择继续-立即安装（或自定义可更改安装位置再安装），等待安装完成，然后点击关闭。

接下来就是神奇的激活过程啦，下载Office Tool Plus|原网页，解压，打开Office Tool Plus，点击激活，

然后在KMS服务器地址栏输入：www.luochenzhimu.com，然后点击设定服务器地址。

然后再点击激活，显示产品激活成功，OK. Let’s see.打开Word，文件-账户，显示如下：

激活成功。

Adobe系列的安装（以Photoshop CC为例）

下载页面来自落尘之木，

点击Photoshop CC 一行的Windows(64-bit)，下载压缩包，解压，打开set-up.exe, 等待安装完毕，注意安装位置。

接下来我们通过补丁激活（放心，此软件无毒）。这里如果你是win10用户，请通过设置-更新与安全-Windows安全-打开Windows Defender安全中心-病毒和威胁防护-”病毒和威胁防护“设置，关闭实时保护。

然后下载这个压缩包，解压，打开amtemu.v0.9.2-painter。

点击然后选择Adobe Photoshop CC 2017（虽然下载的是Photoshop CC 2018，但是没关系）

然后点击后下角的Install，找到你的安装路径，比如我的是C:\Program Files\Adobe\Adobe Photoshop CC 2018

选中amtlib.dll，然后点击打开。

显示WORKING DONE，OK，激活成功。
